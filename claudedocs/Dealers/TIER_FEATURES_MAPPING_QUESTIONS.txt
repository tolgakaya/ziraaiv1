# TIER FEATURE MAPPING - Karar Formu

## Mevcut Tier'lar:
1. Trial (Ücretsiz deneme)
2. S (Small - Küçük çiftlikler)
3. M (Medium - Profesyonel çiftçiler)
4. L (Large - Büyük çiftlikler)
5. XL (Extra Large - Kurumsal)

## Mevcut Feature'lar:
1. messaging - Mesajlaşma (text mesaj) Bunun yanında attachmentlı mesaj da var(resimli vb)
2. voice_messages - Sesli mesaj
3. smart_links - Akıllı link yönetimi
4. advanced_analytics - Detaylı analytics
5. api_access - API erişimi
6. sponsor_visibility - Sponsor profil/logo görünürlüğü
7. priority_support - Öncelikli destek
8. data_access_percentage - Çiftçi verisi erişim yüzdesi
